# kit-cyber-2026
Le guide visuel pour protéger son identité numérique en 2026.  Les gestes simples pour :   - sécuriser ses mots de passe - éviter le phishing - protéger ses données - limiter les risques liés à l’IA - réduire le traçage numérique.  La cybersécurité n’est pas qu’un sujet d’experts. C’est devenu un sujet de société.  À conserver et partager !
